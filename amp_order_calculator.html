💾 Резервная копия данных

Экспортируйте все сохранённые заказы из истории в JSON-файл. Используйте его для переноса данных на другой компьютер или как архивную копию.

📂 Восстановление из резервной копии

Загрузите ранее сохранённый JSON-файл с резервной копией. Существующая история заказов будет дополнена — дубликаты не добавятся.

🗑️ Очистить историю

Удалить все сохранённые заказы из истории. Действие необратимо.

↩ Сбросить правки каталога

Вернуть все товары в каталоге к оригинальным данным из файла — названия, шт/коробку, вес, габариты.

🛃 Данные для статистической формы

Загрузите один или несколько УПД из 1С (.xls). Система автоматически сгруппирует товары по ТН ВЭД кодам и посчитает количество штук, сумму и общий вес по каждому коду.
